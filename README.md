# Student-Marks-Calculation (Java)

## Project Overview
This project is a prototype of a Student Mark Management System developed in Java. It allows educational administrators to manage course structures, track student performance across various components (assignments, tests and projects) and automate the grading process based on specific academic weightage.

## Features
- Add, edit, delete, and browse courses with customizable coursework and final exam weightage.
- Automatically calculates overall marks and assigns letter grades (A+ to F) based on a predefined scale.
- Implemented try-catch blocks and logical checks to prevent negative marks and ensure Student ID validity.
- Saves all course and student records into CSV files upon termination and reloads them on startup to prevent data loss.
- An interactive, loop-driven menu system for seamless navigation.

## Grading Scale

| 75-79 | (A-)  |

| 65-69 | (B)  |

| 56-59 | (C+)  |
| 50-55 | (C)  |
| 45-49 | (C-)  |
| 40-44 | (D)  |
